Canvas and SVG tags
==> Canvas tag:
- Canvas is raster based and composed of pixel. it means As you zoom in on the graphic,
  you will see small dots.
- Give better performance with larger number of objects.
- Poor text rendering capabilities.
- Poor scalability.

- SVG is used to draw 2D Vector based Grhphics in web pages.
- W3C recomands SVG technology only to draw graphics in a web page.
