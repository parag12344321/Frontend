
- The span tag is an inline element. That means, it can be in a single line 
  compair to paragraph tag because the paragraph<p> tag is a block element.
  Block element means it take up the entire width.
  Any element i.e. next will be after it.
-Ex.:
<html>
<head>
    <meta charset='utf-8'>
    <meta http-equiv='X-UA-Compatible' content='IE=edge'>
    <title>Inline and Block element</title>
    <meta name='viewport' content='width=device-width, initial-scale=1'>
    <link rel='stylesheet' type='text/css' media='screen' href='main.css'>
    <script src='main.js'></script>
</head>
<body>
    <!--span tag-->
    <span>This is a inline element.</span>
    <span>With Paragraph.</span>
    <!--Paragraph tag or Block Element-->
    <p>This is a Block element.</p>
    <p>With Paragraph.</p>
</body>
</html>
